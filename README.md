# Meesho Udaan – The Entrepreneur Discovery Engine for Bharat

Meesho Udaan is an Agentic AI platform that helps skilled individuals across Bharat transform their skills into profitable online businesses.

Instead of asking **"What do you want to sell?"**, Meesho Udaan begins with **"What skill do you have?"** and automatically generates a complete business blueprint using a multi-agent AI workflow.

Built for **ScriptedByHer 2.0 – Meesho Hackathon**.

---

## 🚀 Features

- AI Skill Discovery
- Market Intelligence Analysis
- Trend Forecasting
- Product Recommendation
- Pricing Strategy
- Brand Identity Generation
- SEO-Optimized Product Listings
- Business Growth Roadmap
- Complete Business Blueprint Generation

## ⚙️ Running Locally

### Backend

```bash
cd backend

python -m venv venv

# Windows
venv\Scripts\activate

pip install -r requirements.txt

uvicorn app.main:app --reload
```

### Frontend

```bash
cd frontend

npm install

npm run dev
```
